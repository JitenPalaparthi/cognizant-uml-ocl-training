# OCL Keywords

## Introduce constraints
    - inv,pre,post
## Conditional expressions
    - if-then-else-endif
## Boolean operators
    - not,or,and,xor,implies
## packages
    - package,endpackage
## Introduces constraints
    - context
## Global definition
    - def 
## Local definition
    - let, in 
## Attribute derivation
    - derive
## Initial value derifation
    - init
## Result value from an operation
    - result
## Object being consratied
    - self

# Builtin types and literals

- Boolean : true | false
- Integer: +ve and -ve numbers
- Real: real numbers, rational, irreational(decimal) all sorts of numbers
- String: collection of characters (utf-8 char sequence is used)
